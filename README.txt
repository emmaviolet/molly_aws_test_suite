This is a test suite for Molly AWS Gateway API

You need to set up env file on your local machine for the specs to work

** create a file named ~/app-env with the following entries

export MOLLY_AWS_TEST_API_KEY=<AWS TEST API KEY>
export MOLLY_AWS_TEST_URL=<AWS API TEST URL>

** add the following line to your .bash_profile

source ~/app-env
